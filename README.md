# Todo App (Full Stack)

## Features
- User Authentication (JWT)
- Add / Edit / Delete Tasks
- Search Tasks
- Filter & Sort Tasks
- Category (Personal, Work, Urgent)
- Task Status (Completed / Pending)
- Modern UI (Tailwind CSS)

## Tech Stack
- Frontend: HTML, CSS, JavaScript, Tailwind
- Backend: Node.js, Express
- Database: MongoDB

## Setup
1. Clone repo
2. Run backend:
   npm install
   npm start
3. Open frontend
